# Paperplane

Paperplane project description.

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended) or npm

### Installation

```bash
pnpm install
```

### Development

```bash
# Run in development mode with hot reload
pnpm dev

# Build the project
pnpm build

# Run tests
pnpm test

# Type check
pnpm typecheck
```

## Project Structure

```
.
├── src/           # Source code
├── dist/          # Compiled output (git ignored)
├── tests/         # Test files
└── package.json   # Project configuration
```
